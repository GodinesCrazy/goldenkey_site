
# GoldenKey Studios — Sitio actualizado

Este paquete incluye:
- `index.html`: portada con hero y cards de apps.
- `apps.html`: detalle de RoveCast, AstroDía y Voxtide con enlaces a Google Play.
- `ivan_reseller/index.html`: landing para aprobación en AliExpress Portals (usar como URL de canal).
- `privacy.html` y `terms.html`: páginas legales básicas.
- `assets/styles.css`: estilo moderno sin dependencias.

## Personalización rápida
- Cambia el email de contacto en el footer.
- Sustituye los enlaces `#` de Ivan Reseller por tus enlaces firmados con `trackingId` cuando la API esté activa.
- Reemplaza `assets/hero-showcase.png` por un banner real.

## Deploy
1. Sube todo el contenido a Netlify/Vercel o tu hosting.
2. Usa `https://<tu-dominio>/ivan_reseller/` como URL en el formulario de Portals.
